돼지고기김치찌개

돼지고기 김치찌개만이 진리다!!

재료목록:

  • 익은김치 1/2포기
  • 두부 1모
  • 대파 1단
  • 간장 2스푼

조리 순서:

  1. 냄비에 참기름을 두어 바퀴 돌려준 뒤 삼겹살을 넣고 후추 약간과 마늘 한 스푼 간장 2스푼을 넣고
  2. 고기 겉면이 골고루 익을 때까지 볶아준다.
  3. 고기 겉면이 골고루 익었다 싶으면 썰어놓은 김치를 넣고
  4. 김치가 노릇노릇해질 때까지 볶아준다.
  5. 물을 적당량 부어준 뒤 강물로 한번 확 끓여준다
